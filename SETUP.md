-   Install dependencies

```console
$ npm install
$ npm install --workspace=@react-pdf-viewer/nextjs-demo
$ npm install --workspace=@react-pdf-viewer/webpack-typescript-demo
```

-   Build all packages

```console
npm run build
```

-   Remove the built files in all packages:

```console
npm run clean
```

-   Check the formats

```console
npm run lint
```
